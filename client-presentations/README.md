# Клиентские HTML-презентации

Персональные страницы публикуются внутри сайта и отправляются клиенту обычной ссылкой.

## Как создать новую страницу

1. Скопировать папку `template` или готовую папку клиента.
2. Создать папку клиента: `client-presentations/client-name-date/`.
3. Заполнить `data.json` и `index.html` по данным звонка.
4. Проверить, что нет персональных данных клиента: фамилии без необходимости, телефона клиента, источника денег, паспортных данных и чувствительных деталей разговора.
5. Проверить ссылки: телефон, сайт, фрагмент Первого канала.
6. Сделать commit и push.
7. Отправить клиенту ссылку: `https://leontev.pro/client-presentations/client-name-date/`.
