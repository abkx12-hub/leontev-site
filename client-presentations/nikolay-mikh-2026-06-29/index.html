Что сравниваем

Для Николая важна не только цена квартиры. Реальная стоимость = квартира + ремонт + паркинг + риск локации.

Площадь67–72 м²

Маленькие 57–58 м² проходят хуже, даже если цена выглядит красивой.

Кухня17+ м²

Кухня — один из главных фильтров. 11–13 м² для него слабый вариант.

Ремонт100–150 тыс./м²

Голые стены могут добавить 7–12 млн к итоговой стоимости.

Паркинг+2–3+ млн

Без паркинга для него решение неполное: вопрос хранения машины остаётся важным фактором решения.
