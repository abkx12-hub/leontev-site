# Карта сайта leontev.pro

Обновлено: 2026-07-03.

Цель карты — держать сайт как единую воронку: **консьерж-сервис при покупке квартиры → что входит → почему без системы хаос → дорожная карта → цена ошибки → форматы → кейсы → контакт**.

Основной термин сайта: **консьерж-сервис при покупке квартиры**. Черновики, презентации и технические витрины не выводятся в верхнее меню.

## Archive / copy-fix

| Тип | Статус | Что делать |
|---|---|---|
| Старые страницы с `buyer concierge` / `property concierge` | archive | Заменять термином “консьерж-сервис при покупке квартиры”. |
| Старые страницы с `shortlist` как ключевым термином | copy-fix | Заменять на “финальные варианты” или “рабочие сценарии”. |
| “личный штаб” как главное позиционирование | copy-fix | Можно использовать точечно как образ, но основной термин — “консьерж-сервис”. |
| Старые презентационные черновики без маршрута и CTA | archive | Не добавлять в меню и sitemap. |

## Правило верхней навигации

В верхнем меню держим только основные смысловые блоки: `Сервис`, `Хаос`, `Маршрут`, `Кейсы`, `СМИ`, `Форматы`, `Контакты`. Черновики, презентации, тестовые страницы и служебные инфографики не выводятся в главное меню.
