# Изображения для сайта leontev-site

Файлы подготовлены для загрузки в репозиторий в папку `/assets/`.

## Основные файлы

- `project-exterior.jpg` — фон/изображение для блока «Объект / локация».
- `lobby-detail.jpg` — фон/изображение для блока «Качество объекта».
- `city-view.jpg` — фон/изображение для блока «Стратегия / маршрут покупки».

## WebP-дубли

- `project-exterior.webp`
- `lobby-detail.webp`
- `city-view.webp`

Можно использовать `<picture>`:

```html
<picture>
  <source srcset="assets/project-exterior.webp" type="image/webp">
  <img src="assets/project-exterior.jpg" alt="Современный жилой комплекс бизнес-класса">
</picture>
```

## Правила использования

- Не вставлять в первый экран вместо портрета Павла.
- Использовать как вторичный визуальный слой в блоках ниже hero.
- Обязательно затемнять overlay, чтобы текст был читаемым.
- На мобильной версии не перегружать первый экран изображениями.
