# Фотографии для следующего этапа

Добавляйте сюда только реальные изображения, на использование которых есть права.

- `about-photo.jpg` — реальная фотография Павла в деловой среде: на встрече, в офисе или у окна.
- `project-exterior.jpg` — реальный фасад или архитектура жилого проекта.
- `lobby-detail.jpg` — реальное лобби, деталь отделки, панорамный вид или макет проекта.

Пока эти файлы отсутствуют, сайт остаётся цельным: он использует тёмные CSS-градиенты, архитектурные линии и карточки. В `index.html` нет ссылок на эти будущие файлы, поэтому битых картинок не появится.

Не добавляйте `hero-bg.jpg`, `detail-quality.jpg`, `route-bg.jpg` или `cta-bg.jpg` из прежнего набора, если это плоские иллюстрации или изображения с текстом. Первый экран должен оставаться построенным вокруг реального `pavel-photo.jpg`.
