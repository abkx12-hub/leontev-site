# Фотографии для следующего этапа

Добавляйте в эту папку только реальные фотографии, на использование которых есть права.

- `hero-bg.jpg` — широкий архитектурный фон: современный ЖК, город или фасад.
- `about-photo.jpg` — Павел в деловой среде, у окна, в офисе или рядом с современным жилым проектом.
- `detail-quality.jpg` — лобби, фасад, панорамный вид или макет проекта.

Сайт сейчас не зависит от этих файлов: пока они отсутствуют, он использует тёмные архитектурные градиенты и не показывает битые изображения.
